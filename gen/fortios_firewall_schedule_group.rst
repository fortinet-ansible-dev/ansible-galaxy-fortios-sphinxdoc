fortios_firewall_schedule_group -- Schedule group configuration in Fortinet's FortiOS and FortiGate.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify firewall_schedule feature and group category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.14

----

Examples
--------

.. code-block:: yaml+jinja
    
    - name: Schedule group configuration.
      fortinet.fortios.fortios_firewall_schedule_group:
          vdom: "{{ vdom }}"
          state: "present"
          access_token: "<your_own_value>"
          firewall_schedule_group:
              color: "0"
              fabric_object: "enable"
              member:
                  -
                      name: "default_name_6 (source firewall.schedule.onetime.name firewall.schedule.recurring.name)"
              name: "default_name_7"
