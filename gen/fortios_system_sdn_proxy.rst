fortios_system_sdn_proxy -- Configure SDN proxy in Fortinet's FortiOS and FortiGate.
++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++++

.. versionadded:: 2.0.0

.. contents::
   :local:
   :depth: 1


Synopsis
--------
- This module is able to configure a FortiGate or FortiOS (FOS) device by allowing the user to set and modify system feature and sdn_proxy category. Examples include all parameters and values need to be adjusted to datasources before usage. Tested with FOS v6.0.0



Requirements
------------
The below requirements are needed on the host that executes this module.

- ansible>=2.14

Notes

Examples
--------

.. code-block:: yaml+jinja
    
    - name: Configure SDN proxy.
      fortinet.fortios.fortios_system_sdn_proxy:
          vdom: "{{ vdom }}"
          state: "present"
          access_token: "<your_own_value>"
          system_sdn_proxy:
              name: "default_name_3"
              password: "<your_own_value>"
              server: "192.168.100.40"
              server_port: "0"
              type: "general"
              username: "<your_own_value>"
